## Tech Stack
- **Frontend:** React, Vite, TailwindCSS, HeadlessUI
- **Backend:** Node.js, Express, TypeScript, Sequelize (PostgreSQL)
- **Testing:** Vitest (Unit), Playwright (E2E)
- **Infra:** Docker
